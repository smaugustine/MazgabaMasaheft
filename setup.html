Setup

A single folder (e.g. "Beta Masaheft") should be used to contain all of the relevant data.

Within this folder, clone the data repositories you wish to use. The available repositories are:

  • Manuscripts
  • Works
  • Persons
  • Places
  • Institutions

These should not be renamed but rather keep the default name (e.g. "Manuscripts", "Works", etc.).

Only data from cloned repositories will be available. Thus, titles of works, for example, will be unavailable in manuscript records unless both repositories (Works and Manuscripts) are cloned.
